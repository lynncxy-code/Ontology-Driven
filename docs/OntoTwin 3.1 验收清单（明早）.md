## 三、真机验收步骤（建议照走）

1. **坐标标定 → 进入「标定」步骤**：
   - [ ] 看到新的「画布朝向」面板（旋转 0/90/180/270 + 镜像翻转 + 复位）。
   - [ ] 改「旋转 90°」→ 画布整体转 90°（CAD 跟着转，**这就是消除打点脑内旋转**）。
   - [ ] 勾「镜像翻转」→ 画布左右翻。改完右侧出现「已保存」。
   - [ ] 转/翻后**打点**：点画布某设备 → 锚点落在你点的位置（点击坐标已正确反算，不受显示朝向影响）。
2. **完成标定 → 保存构件**：
   - [ ] 正常保存（构件现在带规范坐标，UE 坐标为派生）。
3. **实例绑定台 → 绑定 → 铸造**：
   - [ ] 实例正常出现。
4. **实例运维 → 选一个实例 → 点「微调位置」**：
   - [ ] 弹窗预填该实例的规范 X/Y（mm）与朝向。
   - [ ] 改 X/Y → 应用 → 提示「已更新」→ 该实例位置变化（UE 下一轮轮询生效）。
5. **UE**：
   - [ ] 朝向若仍与现场不符，回坐标标定改「旋转/翻转」即可；**无需改 UE**。

## 四、已知边界 / 后续（非本版，已对齐）

- **楼层表编辑 UI**：后端/API 已支持多楼层（floor→Z），但前端暂无「楼层表」编辑器，构件默认 floor=1。当前 W18 每层独立关卡，单层即可用；多层批量编辑留下一迭代。
- **原点/UE 原点字段 UI**：`canonical_origin` / `ue_origin_cm` 已可经 API 设置，面板暂未暴露（锚点拟合已含平移，故影响小）。
- **海康等外部源**：仅留通用帧接口，未接实时数据（见 `docs/坐标体系 - 海康等外部源待核实项.md`）。
- **UE 根 Actor（整场拖动/多工厂拼关卡）**：B2 方案，留到真需要时。

## 六、若验收发现问题

- 画布转向方向不符直觉（如转的方向反了）：rotation 改成 270° 或勾翻转即可——它是粗对齐，本就靠肉眼调到与 UE 一致。
- 「无激活项目」提示：朝向面板需要先有激活项目（合并过类型库）；纯 demo 下退化为恒等显示，属预期。
