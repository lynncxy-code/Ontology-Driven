## 三、真机验收步骤

### A. 原点（P1）
1. 坐标标定 → 进「标定」步骤 → 朝向面板下方有「规范原点(CAD)」：
   - [ ] 点「在画布点选」→ 画布单击某点 → 出现品红「规范原点」十字标记；X/Y 数值填入；提示「已保存(全场已重算)」。
   - [ ] 也可直接在 X/Y 框输入数值。
   - [ ] origin≠0 时，保存构件后 UE 位置不偏移（组合修正生效）。

### B. 映射目标 + 多目标标定（P3）
2. 标定步骤顶部有「映射目标」选择器（默认 UE）：
   - [ ] 正常标 UE：放锚点、填 UE(cm)、计算变换 → 合格。
   - [ ] 点「+ 新建源」→ 输入名称(如"海康RCS")→ 选择器切到该源，锚点列读数列变为该源(mm)。
   - [ ] 同一批 CAD 点保留，给每个点填该源坐标 → 计算变换 → 提示 RMSE。
   - [ ] 切回 UE，读数列切回 UE 值（CAD 点不变）。
3. [ ] 保存构件正常。

### C. 多坐标联动面板（P2）
4. 实例运维 → 选实例 → 「微调位置」：
   - [ ] 弹出多坐标面板：规范(mm) / UE(cm) 两行（+ 已标定的自定义源行），各 X/Y(/Z) 可编辑。
   - [ ] 改「规范 X」→ UE、源 那行自动联动变化。
   - [ ] 改「UE X」→ 规范、源 自动联动。
   - [ ] 改楼层下拉 → Z 跳到该层基准。
   - [ ] 保存 → 提示「写入规范坐标」；UE 中实例移动到新位置且持久。
5. [ ] Override 面板已无 位置/朝向（T/R 滑块），只剩 可见性/材质/动画/特效。

## 六、若验收发现问题

- 自定义源那列不出现在多坐标面板：确认该源已「计算变换」成功（有 from_canonical 才显示）。
- 改 UE 联动跳动：换算在 @change(失焦)触发，逐栏 round，属正常；若数值漂移过大检查该帧/UE 矩阵是否退化。
- 新建源后选择器没更新：点「刷新」或重进标定步骤（loadCalibFrames 在进入标定时拉取）。
